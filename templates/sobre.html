Flask

O Flask é um microframework web gratuito e de código aberto escrito em Python. É uma estrutura leve e flexível que é adequada para criar aplicações web simples e complexas.

Bootstrap

O Bootstrap é um framework CSS gratuito e aberto que pode ser usado para criar sites e aplicativos web responsivos. Ele fornece uma ampla gama de componentes de interface do usuário, incluindo botões, menus, formulários e muito mais.
